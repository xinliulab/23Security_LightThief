# Citation

If you use this software in academic work, please cite the USENIX Security 2023
LightThief paper:

> *Your Optical Communication Information Is Stolen behind the Wall*.
> USENIX Security Symposium, 2023.

Please obtain the complete author list, page numbers, and publication URL from
the official USENIX paper page or your bibliography database.

When referring specifically to this repository, include the repository URL and
the commit hash or release tag used in your experiment.
